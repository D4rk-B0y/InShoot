## How to Install in Termux

`$ pkg update && upgrade -y`

`$ pkg install openssl-tool`

`$ pkg install tsu`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/D4rk-B0y/InShoot.git`

####For 32 bit####

`$ cd InShoot`

`$sudo pip install -r requirements.txt`

`$python -m pip install requirements.txt`

`$pip3 install -r requirements.txt`

`$ termux-wake-lock`

`$ python InsF_V1.py`

#####For 64 bit#####

`$ cd InShoot`

`$ termux-wake-lock`
